## 목적

v0.1은 `font/family/sans` 시스템 스택을 사용합니다. 크기와 행간은 같은 이름의 단계로 조합하고, 굵기는 의미 있는 위계에만 사용합니다.

## 사용 문법

```css
font-family: var(--ds-font-family-sans);
font-size: var(--ds-font-size-body);
line-height: var(--ds-font-line-height-body);
```
