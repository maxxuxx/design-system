## 목적

Primitive는 실제 팔레트 값이고 Semantic은 배경, 텍스트, 행동, 상태의 역할입니다. 제품 UI는 Semantic 색상만 사용합니다.

## 사용 문법

```css
color: var(--hds-color-text-primary);
background: var(--hds-color-bg-surface);
```
